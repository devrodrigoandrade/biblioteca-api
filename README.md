# Biblioteca API

API REST para gerenciamento de livros desenvolvida com Spring Boot.

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Swagger/OpenAPI

## Endpoints

GET /livros
GET /livros/{id}
POST /livros
PUT /livros/{id}
DELETE /livros/{id}

## Executando o projeto

1. Clone o repositório
2. Configure o MySQL
3. Execute a aplicação

## Swagger

http://localhost:8080/swagger-ui/index.html
